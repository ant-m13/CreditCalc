# Ипотечный кредитный калькулятор

Локальное React-приложение для расчёта ипотеки, досрочных погашений, льготных периодов и сравнения стратегий. Денежные вычисления выполняются через `decimal.js`, расчётное ядро не зависит от UI.

## Запуск

```bash
pnpm install
pnpm dev
```

Приложение откроется на `http://127.0.0.1:4317`.

```bash
pnpm test   # 24 unit-теста
pnpm build  # production-сборка
```

## Поддерживается

- аннуитетные и дифференцированные платежи;
- ежемесячная, двухнедельная и квартальная периодичность;
- ежедневное или периодическое начисление процентов;
- базы 360, 365, 366, Actual/365 и Actual/Actual;
- округление до копеек, рублей и банковское округление;
- досрочное погашение с уменьшением срока, платежа, полным закрытием или комбинированной стратегией;
- процентный первый платёж без погашения основного долга;
- ввод как отдельной досрочной части, так и общей суммы строки из банковского графика;
- точная обработка досрочной даты внутри процентного периода, порядка операций и погашения процентов;
- источники: собственные средства, субсидия/маткапитал, страховое возмещение, прочее;
- льготные периоды: полная отсрочка, только проценты, уменьшенный и индивидуальный платёж;
- комиссии и порог автоматического закрытия малого остатка;
- ввод срока в месяцах или годах с автоматическим преобразованием;
- настраиваемая точность отображения сумм и четыре цветовые схемы;
- RUB, USD, EUR и CNY с корректным форматированием;
- сравнительная аналитика, графики, таблица графика;
- экспорт CSV, Excel-совместимого XLS, JSON и независимый полный отчёт для печати/PDF.

Параметры, сценарии, тема и формат сумм сохраняются локально в браузере.

## Публикация на GitHub Pages

Репозиторий содержит готовый workflow `.github/workflows/deploy-pages.yml`. При каждом push в ветку `main` он запускает тесты, собирает приложение с корректным путём репозитория и публикует папку `dist`.

Для первого запуска откройте в GitHub `Settings → Pages` и в разделе `Build and deployment` выберите источник `GitHub Actions`. Затем запустите workflow `Deploy to GitHub Pages` вручную во вкладке `Actions` или отправьте изменения в `main`.

После успешной публикации сайт этого репозитория будет доступен по адресу:

```text
https://ant-m13.github.io/CreditCalc/
```

Настройки приложения сохраняются в `localStorage` отдельно для этого адреса и не отправляются на сервер.

## Допущения

- расчёт является финансовой моделью, а не банковской офертой; фактический график банка может отличаться из-за внутренних правил операционного дня;
- досрочный платёж отражается отдельным событием в фактическую дату; проценты до и после него считаются раздельно по соответствующему остатку;
- комиссия за досрочное погашение вычитается из суммы, направляемой в основной долг;
- Actual/Actual делит переходящий через Новый год период на части с делителем 365 или 366 для каждого календарного года;
- при остатке ниже заданного порога он включается в последний платёж и кредит закрывается;
- экспорт PDF реализован через оптимизированную печатную версию браузера.

## Структура

`src/loanEngine` содержит типы, расчёт платежа и процентов, генератор графика, досрочные погашения, льготные периоды, сравнение сценариев, округление и валидацию. `src/App.tsx` отвечает за интерфейс, графики и экспорт; `src/store.ts` — за состояние приложения.
